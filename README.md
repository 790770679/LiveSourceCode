## 代码文件夹说明
1. ParallaxRefresh  
 结合[Ultra-Pull-To-Refresh](https://github.com/liaohuqiu/android-Ultra-Pull-To-Refresh)自定义下拉刷新视差动画。
2. HttpRestModule  
 Android Http结合业务请求JavaBean、List<JavaBean>和Map等复杂对象。
3. CoolHttp  
 从0开始设计一个Http框架。
